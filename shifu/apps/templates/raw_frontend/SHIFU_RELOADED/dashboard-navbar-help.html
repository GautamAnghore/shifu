Installing the software
Building the software

Before the software applications can be used, they need to be built from the source. Below it is explained how to perform this process. The user can either download the required software and frameworks from the internet or can go the dump provided with the software and extract the required data.

Start from Scratch

On a appropriate app engine, create a suitable environment with the required dependencies installed

  • Python 2.7
  • MongoDB Server Application
  • Flask Framework
  • PyMongo
  • WTForms
  • Passlib
  • Bleach
Create Users
Add Administrator

On launching Shifu for the very first time, an option of creating an Administrator for the software would be available for which the user would have to fill the appropriate credentials as per the requirements and hit Sign Up button at the bottom.

There can be only one Administrator, who would be given all the privileges from creating a new user to deleting the whole website.

Page Operations
Add Website Name

Once the Administrator has logged in successfully, the administrator would have the right to choose a name for the Website being made.

Add Theme

After an appropriate name has been chosen, the Administrator would be redirected to a page where a list of different themes would be available to choose from for the Website. Once the theme has been chosen, the Administrator would be averted to the Dashboard, where he would be able to add pages, add users, change theme of the Website Website, delete pages etc.

Add Page

The Add Page option is available when the Pages tab on the left of the Dashboard is hovered upon. The administrator would have to fill the name, URL and Page Description and click on Save and Continue button at the bottom.

The Page URL should be carefully chosen as it cannot be changed once the page is saved. Once a page has been successfully added, it will be visible on the Dashboard of Shifu.

Add Content

Once a page has been successfully added, the user is averted to a page where the content for that page can be added.

Edit Page

To edit the name or description of a page go to the Dashboard, select the page which is to be edited and click the Edit Page option available at the bottom right of the page.

As soon as the Edit Page is clicked upon, the user would be redirected to a page where an option to edit the same would be a available.

Edit Content

To edit the content of a page go to the Dashboard Dashboard, select the page whose content is to be edited and click the Edit Content option available at the bottom right of the page.

As soon as the Edit Content is clicked upon, the user would be redirected to a page where an option to edit the content would be available.

Delete Page

The Delete Page option is also available when the Pages tab on the left of the Dashboard is hovered upon. The administrator would have to fill the Page name and Page Description and click on Save and Continue button at the bottom.
